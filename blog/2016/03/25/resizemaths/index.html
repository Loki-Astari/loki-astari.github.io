1) B:   Size of Allocation
2) r:   Multiplier
3) n:   Number of re-sizes

Size of Block after `n` resizes \[ S = Br^n \] Size of all deallocated blocks after `n` resizes \[ \sum_{k=0}^{n-1}\ Br^k \] Solve for `r` where: \[ \sum_{k=0}^{n-1}\ Br^k <= Br^{n+1} \] \[ B * \sum_{k=0}^{n-1}\ r^k <= Br^{n+1} \] \[ \sum_{k=0}^{n-1}\ r^k <= r^{n+1} \] \[ {1-r^{(n-1)+1}\over1-r} <= r^{n+1} \] \[ {1-r^n\over1-r} <= r^{n+1} \] \[ 1-r^n <= r^{n+1} (1-r) \] \[ 1-r^n <= r^{n+1} - r^{n+2} \] \[ 1 + r^{n+2} - r^{n+1} - r^n <= 0 \] \[ 1 + r^n (r^2 - r - 1) <= 0 \] \[ 1 + r^n\Phi <= 0 \]
